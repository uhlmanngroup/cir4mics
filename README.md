## Setup 


### Intallation 
`git clone https://github.com/MariaTheiss/NPCpy.git`  
`cd NPCpy`  

### Install poetry 
`pip install poetry`

### Open a poetry shell
`poetry shell`

### Install dependencies 
`poetry install` 


### Run tutorial 
`jupyter-lab NPCpy/NPC_testlab.ipynb`


### Run minimal script 

- Start an IDE of your choice 
- Load NPCpy/NPCpy/call_functions.py
